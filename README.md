# Software house landing page
Site made from Pagepro layout

## Life demo
http://dariuszbaczyk.pl/portfolio/software-house/

## Get ready
1. Clone repository
2. Type in terminal:
```
npm intall
```
3. Compile:
For development with watch run:
```
npm run dev
```

For development build run:
```
npm run build
```

For production run:
```
npm run production
```
4. Run index.html from dist folder

## Classes legend

Class prefixes

```
"l-" -> layout style
"c-" -> component style
"js-" -> JavaScript hooks
"is-", "has-" -> state classes
"t-" -> typografy style
"t1", "s1" -> typografy sizes
"u-" -> utility classes
```
